# My dotfiles

This directory contains my dotfiles

## Requirements

- git
- stow

## Installation

- clone the remote repository
- move in the dotfiles directory
- create the symlinks using stow
  '''
  $ git clone https://github.com/galactic1106/gnu-stow-dotfiles.git
  $ cd dotfiles
  $ stow .
  '''
